晶片設計與模擬

專注於奈米級系統晶片開發。深入探討高精度時間數位轉換器、電壓與製程變異下，晶片底層電路的精密時序模擬與誤差校正，優化硬體程式碼與系統效能。

// 實驗室_節點_01

嵌入式軟硬體整合

以高效能微控制器為核心，結合感測元件與開源韌體編程。引入數位雙生電路模擬工具進行開發前的數位驗證，大幅縮短複雜硬體電路除錯與打樣週期。

// 實驗室_節點_02

智慧軌道與安全防護

深耕前瞻鐵道機電整合。研發智慧障礙物偵測與通訊連鎖控制技術，實現平交道與車站月台無盲區防護。並針對集電靴供電穩定度進行精密數據監測，確保鐵道系統安全運行。
